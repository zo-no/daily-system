# Daily System — 你的 AI 日记助手

## 这是什么

一个极简的每日记录系统。你只需要**每晚花 5 分钟**填写今天的流水账，AI 帮你整理成完整时间线，自动生成柳比歇夫式复盘报告，第二天早上告诉你今天该做什么。

核心价值：**不需要你写日记，只需要你记流水账**。

---

## 一天的流程

```
08:00  早报
       「昨天核心工作 3h，今天建议先搞定项目文档」

       [白天自由，不用打卡]

22:30  AI 提醒你写日记
       → 打开链接，填写今天的流水账（5 分钟）
       → AI 自动整理时间线
       → AI 生成今日复盘报告发给你
```

---

## 安装（复制这句话发给你的 AI）

```
去 https://github.com/zo-no/daily-system.git 下载并安装日记系统
```

AI 会自动完成所有安装步骤，途中会问你几个问题（几点睡觉、几点起床、生活里最重要的几件事），回答完就好了。

---

## 安装好之后你只需要做这些

1. 收到打卡链接，存到手机桌面
2. 每晚收到提醒时填写（5 分钟）
3. 早上看 AI 发来的今日建议

---

## 常用命令（直接告诉 AI）

| 说什么 | 会发生什么 |
|--------|-----------|
| 「打卡链接」 | 获取今天的打卡网页 |
| 「今天日记」 | 查看今天的记录 |
| 「昨天日记」 | 查看昨天的记录 |
| 「复盘」 | 手动触发今日复盘分析 |

---

## 复盘报告长什么样

每晚写完日记后自动生成，包含：
- **时间分配统计**：今天各类事情花了多少时间
- **模式洞察**：高效时段、时间黑洞
- **身份检查**：对比你的人生目标，今天有没有偏轨
- **明日行动**：一个具体可执行的改变建议
