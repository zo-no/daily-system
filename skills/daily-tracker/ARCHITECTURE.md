# 📓 完美日记系统 - 架构设计

## 核心目标

**让记录变得极简，让反思变得自然**

---

## 整体流程

```
┌─────────────────────────────────────────────────────────────┐
│                      全天分散收集                            │
├─────────────────────────────────────────────────────────────┤
│  🌅 09:00         🌤️ 12:00        🌇 15:00        🌙 21:00 │
│  问目标           问进度          问困难         自动汇总    │
│     ↓               ↓              ↓               ↓        │
│  我记录          我记录         我记录          我处理      │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│                      极简网页打卡                            │
├─────────────────────────────────────────────────────────────┤
│  外网链接 → 填写 → 自动保存 → 确认                           │
│  (指挥官随时可填，不受时间限制)                               │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│                      自动处理流                              │
├─────────────────────────────────────────────────────────────┤
│  1. 存到 daily-tracker/{date}.json                         │
│  2. 生成 Obsidian 格式日记 → journal/日记/                  │
│  3. 可选：调用 diary-builder 完善                           │
│  4. 可选：调用 day-reflection 分析                          │
│  5. 晚间反馈报告                                            │
└─────────────────────────────────────────────────────────────┘
```

---

## Skill 清单

### 1. daily-tracker（核心）
**职责:** 负责收集、打包、分发所有日记数据

**功能:**
- 🌅 早上目标卡
- 🌤️ 中午进度卡  
- 🌇 下午困难卡
- 🌙 晚间汇总卡
- 📥 接收网页打卡数据
- 📤 触发后续处理流程

**数据流:**
```
网页打卡 → daily-tracker → 生成 Obsidian 日记 → 调用分析 → 反馈
```

---

### 2. diary-builder（完善）
**职责:** 帮你完善柳比歇夫时间线

**触发:** 手动或 daily-tracker 调用
**输出:** 补充完整的间歇性记录

---

### 3. day-reflection（分析）
**职责:** 柳比歇夫统计 + 洞察 + 建议

**触发:** 手动或 daily-tracker 调用
**输出:** 时间统计、能力矩阵、改进建议

---

### 4. zeno-diary-builder（我的）
**职责:** 帮我完善我自己的反思日记

**触发:** 手动
**输出:** 更完整的 Zeno 日记

---

## 数据存储

### 原始数据
`memory/daily-tracker/{日期}.json`

### Obsidian 日记
`{obsidian_path}/{YYYY_MM_DD}.md`

### 我的日记  
`{data_path}/{YYYY-MM-DD}.md`

---

## 使用方式

| 操作 | 命令 |
|------|------|
| 打卡 | 打开网页链接 |
| 查看今天 | `/today` 或 `/diary today` |
| 查看昨天 | `/yesterday` |
| 完善时间线 | `/diary-builder` |
| 深度复盘 | `/reflect` 或 `/day-reflection` |
| 完善我的日记 | `/zeno-diary` |
| 晚间报告 | `/nightly` |

---

## 自动触发

通过 cron 或 heartbeat 设置：
- 每天 09:00 - 早间目标
- 每天 12:00 - 中午进度
- 每天 15:00 - 下午困难
- 每天 21:00 - 晚间汇总 + 分析

---

## 统一入口

**只有一个核心 skill:** `daily-tracker`

其他都是它的**可选模块**，按需调用。
