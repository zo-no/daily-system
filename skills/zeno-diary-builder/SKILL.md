# 🤖 Zeno Diary Builder - 完善我的日记

**定位:** 双向完善 - 完善你的 + 完善我的  
**触发:** `/zeno-diary`

---

## 流程

1. 读取我今天的日记
2. 分析不完善的地方（时间线空白、缺少洞察、没写成就等）
3. 生成网页表单让我填
4. 更新我的日记

---

## 分析维度

| 检查项 | 不完善表现 |
|--------|-----------|
| ⏰ 时间线 | 只有"白天"/"晚间" |
| 💭 洞察 | 没有具体感悟 |
| 🎯 成就 | 没记录今天完成的事 |
| 😔 卡点 | 没写遇到的困难 |
| 📚 学习 | 没写学到了什么 |

---

## 使用

```
/zeno-diary
```

或

```
完善我的日记
```

---

## 输出

- 网页表单（让我填）
- 更新到 `memory/YYYY-MM-DD.md`
- 确认反馈

---

## 数据位置

我的日记: `memory/2026-03-10.md`
